Quick Start Guide

Step 1: Download & Install

Download Devscribe from our website and drag it to your Applications folder. Launch the app and grant necessary permissions if prompted.

Step 2: Create Your First Document

Click "New Document" or press Cmd+N. Type "/" to open the command menu and explore all available blocks and features.

Step 3: Explore Features

Try these commands to get started:

  • /code - Insert a code block (choose language)
  • /diagram - Create a diagram with Diagramly
  • /api - Add API documentation block
  • /sql - Insert SQL query block
  • /heading - Add section headings

Step 4: Run Your Code

Click the "Run" button (‚ñ∂) on any code block to execute it. Results appear instantly in the console output area below the code.

Frequently Asked Questions

How do I run code in multiple languages?

Type /code, select your language (JavaScript, TypeScript, Java, Python, or SQL), write your code, and click the Run button or press Cmd+Enter.

Where are my documents saved?

By default, documents are saved locally on your Mac. You can choose the save location when creating a new document. Pro version (coming soon) will offer cloud sync.

Can I export my diagrams as images?

Yes! Right-click on any diagram and select "Export as PNG" or "Export as SVG" to save it as an image file.

How do I use the Sakila database?

Create an SQL block, select "MySQL (Sakila)" as the database, and write your query. The Sakila sample database is pre-configured and ready to use.

What is the @baseUrl feature?

Define @baseUrl = https://api.example.com once in your document, then use {{baseUrl}} in all your API requests. Perfect for switching between dev/staging/production environments.
