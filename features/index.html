Powerful Features for Modern Developers

Everything you need to document, design, and develop in one macOS application

Multi-Language Code Execution

Run code snippets directly inside your documents without any terminal or setup. Execute and test your code instantly with real-time output.

  • JavaScript & TypeScript - Run modern JS/TS code with console output
  • Java - Execute Java programs with full console support
  • Python - Run Python scripts and see results immediately
  • SQL - Query databases with Sakila demo or SQLite
  • Real-time Output - See results instantly in console panel
  • Mix Languages - Combine multiple languages in one document

Perfect for tutorials, testing logic, code demonstrations, and technical documentation with live examples.

// JavaScript Example console.log("Welcome to Devscribe!"); function fibonacci(n) { if (n <= 1) return n; return fibonacci(n-1) + fibonacci(n-2); } console.log(fibonacci(8)); // TypeScript Example type User = { id: number; name: string; }; const user: User = { id: 1, name: "Developer" }; console.log(user);

Diagramly - Visual Design Tool

Built-in diagramming tool for creating professional architecture diagrams, database schemas, and system designs with an intuitive interface.

  • Database ERD Diagrams - Create entity-relationship diagrams with proper cardinality
  • UML Class Diagrams - Document object-oriented architecture and relationships
  • High-Level Design (HLD) - Design system architectures with AWS components
  • DSA Visualizations - Visualize stacks, queues, linked lists, arrays, matrices
  • Drag-and-Drop Interface - Easy to use, similar to draw.io
  • Export as Images - Save diagrams as PNG for presentations

Ideal for software architects, backend engineers, and educators who need to visualize complex systems.

Built-in SQL Database Support

Pre-configured with the Sakila demo database for practicing MySQL queries, plus SQLite support for creating custom databases directly in your documents.

  • Sakila Demo Database - Pre-loaded MySQL sample database
  • SQLite Support - Create and query your own databases
  • Table View - Results displayed in formatted tables
  • JSON View Toggle - Switch between table and JSON formats
  • Multiple Queries - Run CREATE, INSERT, SELECT statements
  • Perfect for Learning - Practice SQL with real databases

Essential for database designers, backend developers, and students learning SQL.

-- Sakila Database Query SELECT c.first_name, c.last_name, f.title FROM customer c JOIN rental r ON c.customer_id = r.customer_id JOIN inventory i ON r.inventory_id = i.inventory_id JOIN film f ON i.film_id = f.film_id LIMIT 5; -- SQLite Example CREATE TABLE employees ( id INTEGER PRIMARY KEY, name TEXT, department TEXT ); INSERT INTO employees (name, department) VALUES ('Alice', 'Engineering'); SELECT * FROM employees;

API Documentation & Testing

Document and test REST APIs directly within your documents. No need for Postman or external tools. Support for environment variables and multiple request types.

  • Environment Variables - Define @baseUrl once, use everywhere
  • All HTTP Methods - GET, POST, PUT, DELETE, PATCH
  • Path Parameters - Dynamic URL parameters support
  • Query Strings - Add query parameters easily
  • Custom Headers - Set authorization and custom headers
  • Response Formatting - View JSON or table format

Perfect for API documentation, testing endpoints, and maintaining service documentation alongside code.

// Define base URL once @baseUrl = https://api.example.com // Use in requests GET {{baseUrl}}/users/{user_id} // Response (auto-formatted) { "id": 1, "name": "John Doe", "email": "john@example.com" } // Switch environments easily @baseUrl = https://dev-api.example.com

Rich Text Documentation

Create comprehensive technical documentation with support for headers, paragraphs, images, videos, tables, code blocks, and more. Everything in one unified document.

  • Markdown Support - Use familiar markdown syntax

  • Tables & Lists - Organize information clearly
  • Code Blocks - Syntax highlighting included
  • Export Options - Save as .ds format with all content
  • Executable Docs - Mix text with live code and diagrams

Ideal for technical writers, documentation teams, and developers who value comprehensive documentation.

  • Trees & Graphs - Binary trees and graph structures
  • Custom Labels - Add your own data and descriptions

Essential for educators, students, and anyone teaching or learning algorithms and data structures.
