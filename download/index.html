Installation Steps

1

Download the DMG File

Click the download button above to get the Devscribe installer. The file will be saved to your Downloads folder (approximately 120 MB).

2

Open the DMG File

Locate the downloaded file in your Downloads folder and double-click to open it. A new window will appear with the Devscribe application icon.

3

Drag to Applications

Drag the Devscribe icon to the Applications folder shortcut in the installer window. This will install Devscribe on your Mac.

4

Launch Devscribe

Open your Applications folder (or use Spotlight search by pressing Cmd + Space and typing "Devscribe") and double-click the Devscribe icon to launch the application.

5

Grant Permissions (if prompted)

If macOS asks for permissions, click "Open" to confirm. You may need to go to System Preferences > Security & Privacy to allow the app on first launch.

6

Start Creating!

You're all set! Start by creating your first document. Press "/" to explore all features including code execution, diagramming, and API testing.

Download FAQ

Is Devscribe really free?

Yes! Devscribe is completely free to download and use. All core features including code execution, diagramming, and API testing are available in the free version. Pro features with cloud sync will be available in the future.

Which macOS versions are supported?

Devscribe requires macOS 11.0 (Big Sur) or later. It works on both Intel and Apple Silicon (M1/M2/M3) Macs. We recommend macOS 13.0 or later for the best experience.

How much disk space does Devscribe need?

The download is approximately 120 MB. We recommend having at least 500 MB of free disk space for installation and document storage. For optimal performance, 1 GB or more is recommended.

Do I need an internet connection?

No! Devscribe works completely offline. All code execution, diagramming, and documentation features work without an internet connection. You only need internet for API testing features.

How do I update Devscribe?

Devscribe will automatically notify you when updates are available. You can also check for updates manually from the application menu: Devscribe > Check for Updates.

Can I install Devscribe on multiple Macs?

Yes! You can install Devscribe on as many Macs as you own. Your documents are stored locally by default, so you can work on different machines independently.

Is there a Windows or Linux version?

Currently, Devscribe is only available for macOS. We're focusing on delivering the best possible experience on macOS first. Windows and Linux versions may be considered based on user demand.

What if I encounter installation issues?

If you experience any problems during installation, please check our troubleshooting guide or contact our support team.
